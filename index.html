This

This is a small collection of examples of using the Trello Api to build mini-apps which extend and enhance Trello.

You

You have some minimal experience of building a Javascript / HTML app, even if that's only copying and pasting an example. I encourage you to fork this app and hack away.

However, before you do anything else:

  • read these: main docs, API reference, Trello board for API
  • open your browsers' dev console. You'll get a lot more value that way.
